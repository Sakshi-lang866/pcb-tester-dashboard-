# Smart PCB Tester Dashboard

## Overview
This project is a web-based dashboard for monitoring PCB testing using Raspberry Pi.

## Features
- Real-time data monitoring
- Voltage, Current, Temperature display
- Pass/Fail detection
- Graph visualization
- Database storage

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js
- Database: SQLite

## Data Flow
Sensors → Raspberry Pi → Backend → Database → Frontend
